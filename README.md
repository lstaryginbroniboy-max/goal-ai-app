# 🎯 Goal AI Coach

Мобильное приложение — персональный ИИ-коуч, который помогает ставить и достигать целей.

## Что умеет

- **Ежедневный чек-ин** — каждое утро ИИ спрашивает о прогрессе и даёт задачи на день
- **Цели на всё** — отслеживание целей на день / неделю / месяц / год / 5 лет
- **Память** — ИИ помнит всю историю разговоров и подстраивает советы под тебя
- **Задачи** — конкретные ежедневные практики для достижения целей
- **Бесплатный ИИ** — работает с DeepSeek или Groq (оба бесплатны)

## Быстрый старт

### 1. Установи зависимости

```bash
npm install
```

### 2. Получи бесплатный API ключ

**Вариант A — DeepSeek** (почти бесплатно, ~$0.14/1M токенов):
1. Зарегистрируйся на [platform.deepseek.com](https://platform.deepseek.com)
2. Создай API ключ в разделе API Keys
3. Ключ начинается с `sk-`

**Вариант B — Groq** (полностью бесплатно):
1. Зарегистрируйся на [console.groq.com](https://console.groq.com)
2. Создай API ключ
3. Ключ начинается с `gsk_`

### 3. Запусти приложение

```bash
npx expo start
```

Отсканируй QR-код приложением **Expo Go** (скачать в App Store / Google Play).

### 4. Введи API ключ в настройках

При первом запуске откроется онбординг — введи API ключ и свои цели.

## Технологии

- **React Native + Expo** — кроссплатформенное мобильное приложение
- **DeepSeek / Groq API** — ИИ с памятью разговора
- **AsyncStorage** — локальное хранение истории и целей (приватность)
- **TypeScript** — типизация

## Структура

```
app/
  (tabs)/
    index.tsx     — Главная: задачи дня + чек-ин
    chat.tsx      — Чат с коучем
    goals.tsx     — Управление целями
  onboarding.tsx  — Настройка при первом запуске
  settings.tsx    — Настройки (API ключ)
services/
  ai.ts           — Запросы к DeepSeek/Groq с историей
  storage.ts      — Сохранение данных на устройстве
components/
  ChatBubble.tsx  — Пузырь сообщения
  TaskItem.tsx    — Задача с чекбоксом
  GoalCard.tsx    — Карточка цели
```

## Запуск на Android/iOS

```bash
npm run android   # Запуск на Android устройстве/эмуляторе
npm run ios       # Запуск на iOS (только macOS)
npm run web       # Запуск в браузере
```
